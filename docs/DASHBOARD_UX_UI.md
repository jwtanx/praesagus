# Dashboard UX/UI Specification

Status: Draft

Primary Goals

- Surface top trends and their evidence quickly.
- Allow users to drill into data sources, time windows, and supporting records.
- Provide AI copilot integration and natural-language queries.

Key Views

- Home / Trends Feed: ranked signals with short summaries and confidence.
- Entity Page: full timeline, source breakdown, sentiment, velocity charts, and evidence list.
- Watchlists: user-defined entity lists with alerts and historical comparisons.
- Alerts & Subscriptions: manage and review triggered alerts.
- Copilot Panel: persistent chat assistant with context about current entity/page.

UI Principles

- Minimal, data-dense layout.
- Charts: interactive, annotated with evidence markers.
- Provide direct links to raw evidence objects where licensing permits.
