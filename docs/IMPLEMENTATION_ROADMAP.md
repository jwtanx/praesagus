# Implementation Roadmap (MVP → v1 → v2)

Status: Draft

MVP (3 months)

- Implement connector SDK and 2-3 connectors (Reddit, Google Trends, YouTube).
- Store raw in S3; implement Bronze normalization.
- Compute simple signals and expose via API + minimal UI.
- Add alerting and basic backtesting harness.

v1 (6 months)

- Add feature store, model registry, and more connectors.
- Improve Dashboard UI and AI Copilot integration.
- Add production-grade monitoring and RBAC.

v2 (12+ months)

- Full knowledge graph, advanced AI agents, trading integration, and large-scale backtesting.
