# Smart Beta & Factor Investing Skill

## Objective
Deliver a disciplined factor-investing research note that evaluates factor
premia, portfolio construction, performance drivers, and risk exposures for
rule-based or quantitative strategies.

## Outputs
- Factor thesis and definition
- Historical and current factor performance analysis
- Portfolio construction guidance
- Risk exposures and correlation analysis
- Factor implementation recommendation

## Required Sections
1. Executive summary
2. Factor definitions and rationale
3. Historical performance context
4. Current macro and factor regime analysis
5. Portfolio implementation and weights
6. Risk and diversification considerations
7. Recommendation and monitoring plan

## Research Rules
- Define factor rules clearly: value (P/B, P/E), momentum, quality, volatility,
  dividend yield, growth, or size.
- Use historical factor behavior and regime dependencies as evidence.
- Identify where the factor may be crowded or underperforming.
- Explain implementation vehicles: ETFs, quant sleeves, multi-factor products,
  or custom portfolios.
- Include rebalancing frequency and risk-control guardrails.

## Output Template
- Factor: momentum / value / quality / low volatility / dividend / multi-factor.
- Current signal: attractive / neutral / unattractive.
- Recommended allocation: core / satellite / tactical.
- Monitoring triggers: valuation, macro shift, volatility expansion.
